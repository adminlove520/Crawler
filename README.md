# Crawler

## 一. 介绍
     - 学爬虫写的一些小demo,以及在挖漏洞时写的一些收集信息的爬虫，比如爬取必应和百度的搜索结果，各大高校的漏洞信息爬取，子域名暴破等等
     - 使用到的技术
        - request发送HTTP请求
        - 解析数据用到了xpath,re,bs4,jsonpath
        - 无头浏览器
        - 爬虫代理的简单使用
        - 多线程技术
        - 队列的使用
        - 异步爬取简单使用
        - 面向对象
        
## 二.使用说明
    - 所有的爬虫都是使用python3编写的
    - 安装必要的模块即可正常运行
